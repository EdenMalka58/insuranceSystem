Insurance System – Backend Documentation
========================================

Welcome to the backend documentation for the **Insurance System**.

This documentation describes the AWS Lambda–based backend, including:
- API Gateway endpoints
- Lambda handler functions
- Shared Lambda Layers
- OpenAPI request and response schemas
- Authentication and authorization mechanisms

The system is designed following AWS best practices and uses
Amazon API Gateway, AWS Lambda, and Amazon Cognito.
